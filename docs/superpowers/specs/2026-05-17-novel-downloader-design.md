# 小说下载阅读器设计文档

## 1. 项目概述

开发一个 Python 桌面 GUI 小说下载阅读器，支持：
- 从小说网站下载章节内容
- 记录已下载章节，支持增量更新
- 导出 TXT 格式
- 简单预览功能

## 2. 模块结构

```
novel_app/
├── main.py          # 主入口
├── gui.py           # GUI 界面 (tkinter)
├── downloader.py    # 下载器（并发下载）
├── parser.py        # HTML 解析（XPath）
├── database.py      # 数据库操作（SQLite）
├── exporter.py      # 导出器（TXT 导出）
├── config.py        # 配置管理（网站规则）
├── sites.json       # 网站规则配置文件
└── novels.db        # SQLite 数据库（运行时生成）
```

## 3. 网站规则配置

配置文件：`sites.json`

```json
{
  "sites": [
    {
      "name": "新笔趣阁",
      "domain": "xbqg8.net",
      "encoding": "utf-8",
      "catalog": {
        "start_marker": "### 全部章节",
        "end_marker": "本站所有小说为转载作品"
      },
      "content": {
        "start_marker": null,
        "end_marker": "本章未完，请翻下一页继续阅读",
        "ad_patterns": [
          "看正@版.*?章节.*",
          "最新章节首发.*?阅读。",
          "<[^>]+>"
        ]
      },
      "chapter_url_pattern": "/\\d+/\\d+/\\d+\\.html"
    }
  ]
}
```

**配置字段说明：**
- `name`: 网站名称
- `domain`: 网站域名（用于匹配）
- `encoding`: 网页编码
- `catalog`: 目录页解析规则
  - `start_marker`: 目录列表开始标志
  - `end_marker`: 目录列表结束标志
- `content`: 正文页解析规则
  - `start_marker`: 正文开始标志（null 表示从页面主体开始）
  - `end_marker`: 正文结束标志
  - `ad_patterns`: 广告移除正则列表
- `chapter_url_pattern`: 章节链接匹配正则

## 4. 数据库结构

### 4.1 novels 表（小说信息）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |

| author | TEXT | 作者 |
| catalog_url | TEXT | 目录页 URL |
| site_name | TEXT | 来源网站 |

| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 4.2 chapters 表（章节内容）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| novel_id | INTEGER | 关联小说 ID（外键） |
| chapter_index | INTEGER | 章节序号 |
| title | TEXT | 章节标题 |
| url | TEXT | 章节 URL |
| content | TEXT | 章节内容 |
| downloaded | BOOLEAN | 是否已下载 |
| created_at | DATETIME | 创建时间 |

## 6. 核心流程

### 6.1 添加小说流程

1. 用户点击"添加小说"按钮
2. 弹出对话框，输入目录页 URL
3. 根据域名匹配网站规则配置
4. 下载目录页，解析章节列表
5. 保存小说信息和章节列表到数据库
6. 刷新小说列表显示

### 6.2 下载章节流程

1. 用户选择小说，点击"更新"或"下载"
2. 查询未下载的章节列表
3. 使用线程池（ThreadPoolExecutor）并发下载章节内容
4. 解析正文，移除广告
5. 保存到数据库，更新下载进度
6. 刷新界面显示

### 6.3 导出 TXT 流程

1. 用户选择小说，点击"导出 TXT"
2. 弹出文件保存对话框，选择保存路径
3. 按章节顺序读取内容
4. 生成 TXT 文件格式：
   ```
   书名
   作者：xxx
   
   ========== 第一章 水下浮尸 ==========
   
   正文内容...
   
   ========== 第二章 吃人棺材 ==========
   
   正文内容...
   ```

## 8. 错误处理

- 网络请求失败：显示错误提示，支持重试
- 网站规则不匹配：提示用户添加或修改配置
- 章节下载失败：标记失败章节，支持单独重试
- 数据库操作失败：显示错误信息，记录日志

## 9. 并发控制

- 默认并发数：5 个线程
- 支持在配置中调整并发数
- 下载过程中显示进度条
- 支持暂停/继续下载
